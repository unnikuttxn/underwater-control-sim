# AUV Depth Control Simulation

A professional Python implementation of Autonomous Underwater Vehicle (AUV) depth control using a PID controller.

## Features
- PID depth control
- Ocean-current disturbance model
- Euler integration dynamics
- Overshoot and steady-state error metrics
- Response visualization

## Quick Start

```bash
git clone <repo-url>
cd auv-depth-control-professional

python -m venv .venv
.venv\Scripts\activate

pip install -r requirements.txt
python main.py
```
